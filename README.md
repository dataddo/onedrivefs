# onedrivefs

![onedrivefs logo](onedrivefs-logo.svg)

This module implements [`fs.FS` interface](https://pkg.go.dev/io/fs#FS)
introduced in Go 1.16 for
accessing [Microsoft OneDrive](https://onedrive.live.com/about/en-us/) files.
